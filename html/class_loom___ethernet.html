Detailed Description

InternetPlat built off of SSLClient running over an Ethernet Featherwing.

Attention
Requires 7KB of free SRAM at runtime to use.
Resources

  • Product Page: Adafruit Ethernet Featherwing
  • Dependency: EthernetLarge OPEbS Lab fork of Arduino Ethernet library
